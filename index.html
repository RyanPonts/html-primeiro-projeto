Trojan, ou cavalo de Troia, é um tipo de software malicioso que é projetado para parecer inofensivo ou legítimo, mas na verdade é projetado para realizar ações maliciosas em um sistema de computador ou rede. O nome "cavalo de Troia" vem da lenda grega do cavalo de madeira usado para enganar os habitantes de Troia, permitindo que os invasores entrassem na cidade e a conquistassem. Um trojan geralmente é disfarçado como um arquivo ou programa inofensivo, como um jogo ou um utilitário de sistema. Quando o usuário executa o arquivo ou instala o programa, o trojan é ativado e pode realizar várias ações maliciosas, como roubar informações pessoais, controlar o computador remotamente, exibir anúncios indesejados, baixar outros malwares ou até mesmo danificar o sistema operacional. Os trojans são frequentemente distribuídos através de downloads de software não confiáveis, e-mails de phishing ou sites maliciosos. É importante ter um bom software antivírus e manter seus sistemas operacionais e aplicativos atualizados para proteger-se contra trojans e outros tipos de malware.
